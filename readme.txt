How to successfully build this application:
1) File -> Import Module... -> <android-sdk>\extras\google\google_play_services\libproject\google-play-services_lib
2) Ctrl + Shift + Alt + S -> Modules -> <Project Name> -> Dependencies -> Add -> Module Dependency -> google-play-services_lib
3) Add -> Jars or directories -> <android-sdk>\extras\google\google_play_services\libproject\google-play-services_lib\libs
4) Add -> Jars or directories -> <android-sdk>\extras\android\support\v4
